### Install Python 3

* Buka `Terminal`

* Install `Homebrew`

```shell
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew
/install/master/install)"
```

* Pindahkan `PATH` 

```shell
export PATH=/usr/local/bin:/usr/local/sbin:$PATH
```

* Install `Python 3`

```shell
$ brew install python3
```

* Install `Pip` 

```shell
$ install pip
```
